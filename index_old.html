Projects

Forbin Dataset

Forbin dataset is a collection of digitized photographs of the French journalist Victor Forbin, taken between 1890 and 1910. The dataset is composed of 62,135 photographs. Each print underwent digitization on both the front and back sides.


A subset can be explored here or can be downloaded from Hugging Face
Image colorization

Photographic colorization is the process of adding color to black-and-white photos, using modern Machine Learning models based on neural networks. Applications include art, historical restoration, and personal projects. This project, part of the MLOps curriculum at Datascientest, offers an AI-powered photo colorization service.


Image colorization code available on github
