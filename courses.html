Descritpion


Le traitement d'image numérique est un domaine vaste et intéréssant. Acctuellement, la quantité de ce type de données a atteint un record incroyable en vue de la vulgarisation des capteurs optique. Dans ce cours, l'objectif principal est de faire apprendre les bases du traitement d'images aux étudiants. Puis, cet apprentissage est utilisé pour réaliser un projet

Astuces:

Creation d'une matrice inialiser à zeros de type T où T est un des types présenté précédements. Ce dernier peut être un entier (valeur de la case dans le tableau) ou le FLAG associer au type voulu. Par example: CV_8UC3 ou l'entier 16. 
Mat::zeros(nb_lignes, nb_cols, T)
Acceder à un pixel gris (dans le cas ou l'image est de type CV_8UC1). 
(int) img.at<uchar>(i,j)
Acceder aux values d'un pixel couleur de type CV_8UC3. Utilisation du type Vec3b 
Vec3b intensity = img.at<Vec3b>(i,j)
uchar blue = intensity.val[0];
uchar green = intensity.val[1];
uchar red = intensity.val[2];
Acceder aux values d'un pixel couleur de type CV_32FC3. Utilisation du type Vec3f 
Vec3f intensity = img.at<Vec3f>(i,j);
float blue = intensity.val[0];
float green = intensity.val[1];
float red = intensity.val[2];
Remarque: OpenCV lit les images en BGR et non pas en RGB Convertir une image couleur en image grise 
cvtColor(img_source, img_destination, cv::COLOR_BGR2GRAY);
Convertir une image de type UINT8 au type FLOAT 
img_source.convertToimg_destination, CV_32F);
